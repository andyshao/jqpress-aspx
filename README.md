# 博客程序架构
本博客程序是博主11年的时候参考loachs小泥鳅博客内核开发的.net跨平台博客cms，距今已有6年多了，个人博客网站一直在用，虽然没有wordpress那么强大，但是当时在深究.net的同时，自己能写一个博客程序，并且基于独立Linux服务器搭建一个自己的.net网站还是挺有意思。
## 技术栈
- .net framework 4.0
- sqlite 数据库
- mono linux 运行环境以及mono下的sqlite库
- nvelocity 模板引擎
- dapper 轻量级orm框架
- vs2010
